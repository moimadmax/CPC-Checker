    Favoris

    Fonctionnement

    Si activé dans les préférences, les favoris s'affichent sur le popop dans la section "Favoris". Les noms des favoris sont dérivés de leur URL (EnCNCpCOeF pour http://forum.canardpc.com/threads/56458-Extensions-navigateurs-CanardPC-Notifier-Checker-pour-Chrome-Opera-et-Firefox) cependant une info-bulle apparaît en cas de survol du pointeur de souris.

    Le symbole ⇋ à côté du titre s'affiche si la synchronisation est active.

    Ajouter / enlever

    Pour ajouter ou enlever une discussion des favoris, il suffit de se rendre sur la page de la discussion, d'ouvrir le popup et de cliquer sur la puce dans la section "Ajouter/Enlever la page courante" qui apparaît en bas de popup. Il est aussi possible d'ajouter ou d'enlever les discussions qui apparaissent dans la section "nouveautés" en cliquant sur leur puces respectives.
